Declaración de Variables en Python

Descripción: En Python, puedes declarar variables simplemente asignándoles un valor. No es necesario especificar el tipo de datos.

Ejemplo: x = 5 asigna el valor 5 a la variable x.

Ciclo for en Python

Descripción: Un ciclo for se utiliza para iterar sobre una secuencia (como una lista, tupla, rango, etc.) o cualquier objeto iterable.

for elemento in secuencia:
            # Cuerpo del ciclo

Ejemplo:

for i in range(5):
            print(i)

Ciclo while en Python

Descripción: Un ciclo while se ejecuta mientras una condición dada sea verdadera.

while condición:
            # Cuerpo del ciclo

Ejemplo:

x = 0
        while x < 5:
            print(x)
            x += 1

Funciones en Python

Descripción: Las funciones son bloques de código reutilizable que realizan una tarea específica.

Ejemplo:

def saludar(nombre):
            print("Hola,", nombre)
        
        saludar("Juan")

Listas en Python

Descripción: Las listas son estructuras de datos que pueden contener múltiples elementos.

Ejemplo:

mi_lista = [1, 2, 3, 4, 5]
        print(mi_lista[0])  # Imprime el primer elemento

Condicionales en Python

Descripción: Las estructuras condicionales permiten ejecutar cierto código solo si se cumple una condición.

Ejemplo:

if x > 5:
            print("x es mayor que 5")
        elif x == 5:
            print("x es igual a 5")
        else:
            print("x es menor que 5")

Operadores en Python

Descripción: Los operadores son símbolos especiales que realizan operaciones en variables y valores.

Ejemplo:

x = 5
        y = 3
        suma = x + y
        print(suma)

Funciones integradas en Python

Descripción: Python proporciona muchas funciones integradas que están disponibles para su uso sin la necesidad de importar ningún módulo.

Ejemplo:

nombre = "python"
        print(len(nombre))  # La función len() devuelve la longitud de la cadena

Manejo de excepciones en Python

Descripción: Las excepciones son errores detectados durante la ejecución. Python tiene muchas funciones para manejar estas situaciones.

Ejemplo:

try:
            num = int(input("Ingrese un número: "))
            print("El doble del número es:", num * 2)
        except ValueError:
            print("Error: ¡Ingrese un número válido!")

Comprensión de listas en Python

Descripción: La comprensión de listas proporciona una forma concisa de crear listas en Python basadas en secuencias existentes.

Ejemplo:

pares = [x for x in range(10) if x % 2 == 0]
        print(pares)

Diccionarios en Python

Descripción: Los diccionarios son una estructura de datos que permite almacenar pares clave-valor de manera eficiente.

Ejemplo:

diccionario = {'nombre': 'Juan', 'edad': 30}
        print(diccionario['nombre'])  # Imprime el valor asociado a la clave 'nombre'

Operaciones con cadenas en Python

Descripción: Python proporciona numerosas operaciones y métodos para trabajar con cadenas de caracteres.

Ejemplo:

cadena = "Python es genial"
        print(cadena.upper())  # Convierte la cadena a mayúsculas
