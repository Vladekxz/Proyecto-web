1. Completa el siguiente ciclo for en Python:

for i in _____(1, 6):
    print(i)

Resultado esperado de `i`: 1, 2, 3, 4, 5

2. Dado el siguiente ciclo for en Python, completa la fórmula para obtener el valor correcto:

total = 0
for i in range(1, 6):
    total _____ i
print(total)

Resultado esperado: 15

3. Dada la siguiente declaración de variable y ciclo for en Python, completa la fórmula para obtener el valor correcto:

total = 1
for i in range(1, 6):
    total _____ i
print(total)

Resultado esperado: 120

4. Dado el siguiente ciclo while en Python, completa la fórmula para obtener el valor correcto:

total = 1
i = 1
while i <= 5:
    total _____ i
    i += 1
print(total)

Resultado esperado: 120

5. Escribe una función en Python llamada `suma_pares` que reciba una lista de números y devuelva la suma de todos los números pares en la lista.

Por ejemplo:

nums = [1, 2, 3, 4, 5, 6]
        print(suma_pares(nums))

Resultado esperado: 12

6. Escribe una función en Python llamada `conteo_vocales` que reciba una cadena y devuelva un diccionario con el conteo de cada vocal (a, e, i, o, u) en la cadena (sin distinción entre mayúsculas y minúsculas).

Por ejemplo:

texto = "Hola Mundo"
        print(conteo_vocales(texto))

Resultado esperado: {'a': 1, 'e': 0, 'i': 0, 'o': 2, 'u': 1}

7. Escribe una función en Python llamada `es_palindromo` que reciba una cadena y devuelva True si la cadena es un palíndromo (es decir, se lee igual de izquierda a derecha que de derecha a izquierda) y False en caso contrario.

Por ejemplo:

cadena = "anita lava la tina"
        print(es_palindromo(cadena))

Resultado esperado: True

8. Escribe una función en Python llamada `invertir_lista` que reciba una lista y devuelva una nueva lista con los elementos en orden inverso.

Por ejemplo:

original = [1, 2, 3, 4, 5]
        print(invertir_lista(original))

Resultado esperado: [5, 4, 3, 2, 1]

9. Escribe una función en Python llamada `invertir_palabras` que reciba una cadena de texto y devuelva la cadena con todas las palabras invertidas. Por ejemplo, si se recibe la cadena "Hola mundo", la función debería devolver "aloH odnum".

10. Escribe una función en Python llamada `conteo_caracteres` que reciba una cadena de texto y devuelva un diccionario con el conteo de cada carácter en la cadena.

Por ejemplo:

texto = "Hola mundo"
        print(conteo_caracteres(texto))

Resultado esperado: {'H': 1, 'o': 2, 'l': 1, 'a': 1, ' ': 1, 'm': 1, 'u': 1, 'n': 1, 'd': 1}
